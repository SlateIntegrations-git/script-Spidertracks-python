# Spidertracks-Python-Script
Python script to fetch aircraft locations from the Spidertracks API and then convert to Cursor-on-Target (CoT)
See Confluence -- "Adding Spidertracks API to TAK server" for the rest of the installtion instructions.


# Install the Script

# Rocky/Rhel
      sudo dnf install git
      sudo clone https://github.com/SlateIntegrations-git/Spidertracks-Python-Script.git
# Ubuntu
      sudo apt install git
      sudo clone https://github.com/SlateIntegrations-git/Spidertracks-Python-Script.git
